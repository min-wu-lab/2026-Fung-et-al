# Mitotic Wave Analysis
This repository contains analysis code for the manuscript: Mitotic Cdc42 Waves Encode PI(3,4)P2 Signaling and Golgi Morphological State to Control Spindle Scaling.
# Requirement
MATLAB, and [PYthon Microscopy Environment (PYME)](https://github.com/python-microscopy/python-microscopy)

Raw data for Fig 3H,I and others can be found on [Dryad](https://datadryad.org/submission/427522?files)
